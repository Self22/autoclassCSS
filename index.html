AutoclassCSS — Генератор CSS-каркаса

AutoclassCSS — это инструмент для формирования каркаса CSS-классов на основе HTML-разметки по БЭМ-методологии.
Оформление каркаса настраивается по вкусу, благодаря удобному API.
